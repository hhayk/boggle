Generator
	From boggle-generator directory run index.html and export boards via `Save Text to File` button. Document will be stored in ~/Dowloads folder.
	You can change number of generated boards in boggle.js line 4 -> var upperLimit = 10.
Solver
	In terminal from boggle-solver directory run
		`javac Boggle.java`
		`java Boggle /Users/hhayk/Downloads/boggle.txt` with your Downloads path.
